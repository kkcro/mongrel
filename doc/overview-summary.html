Introduction

MongrelMapper is an OTP application to map Erlang records to MongoDB documents. Mongrel has a similar API to the official MongoDB driver but create, read, update and delete operations take record arguments instead of MongoDB tuples.

The benefits of using a mapper and records instead of documents are
  • More expressive code.
  • That the compiler can check that only expected (named) fields are present.
  • That associations between a document and nested child documents can be handled transparently.
